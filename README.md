# Customer Churn Analytics & ML Dashboard

## Objetivo
Analizar el abandono de clientes y construir un sistema predictivo que permita identificar clientes con alta probabilidad de baja.

## Tecnologías
Python, Pandas, Scikit-learn, XGBoost, SHAP, SQL, Streamlit, Matplotlib/Plotly.

## Flujo del proyecto
1. Limpieza y preparación de datos.
2. Análisis exploratorio.
3. Consultas SQL para KPIs de negocio.
4. Entrenamiento de modelos.
5. Evaluación.
6. Interpretabilidad.
7. Dashboard interactivo.

## Resultados principales
- Mejor modelo: XGBoost / Random Forest.
- Métrica principal: Recall y ROC-AUC.
- Principales factores de abandono: contrato, antigüedad, gasto mensual, método de pago.
